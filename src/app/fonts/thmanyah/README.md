# Thmanyah Sans (Arabic UI font)

The Arabic interface uses **Thmanyah Sans**, the free typeface from Thmanyah
(font.thmanyah.com, free for personal and commercial use including websites and app
interfaces). The weight files here are self-hosted and loaded with `next/font/local`
in `src/app/layout.tsx` as the `--font-arabic` variable, which `globals.css` applies to
the whole UI whenever the language is Arabic.

Weights present (woff2):

- `thmanyahsans-Light.woff2` (300)
- `thmanyahsans-Regular.woff2` (400)
- `thmanyahsans-Medium.woff2` (500)
- `thmanyahsans-Bold.woff2` (700)
- `thmanyahsans-Black.woff2` (900)

The full family, the other styles (Serif Display, Serif Text), and the license live in
the `Thmanyah-Font-Family/` folder at the repo root.
